第一阶段：


1. HTML基础语法、HTML常用标签、HTML的语义化、HTML编码规范；
2. CSS 基础语法、选择器、页面布局、排版、盒模型，CSS Hack和 CSS 优化，CSS高级技巧、CSS编码规范；
3. JavaScript语法基础、流程控制、常用内置对象、数据结构、函数和对象、DOM操作、BOM操作、特效；
4.CSS3新增选择器、CSS3动画、CSS3新特性、HTML5新增标签、HTML5新增API（多媒体、拖拽、web存储、定位等）。
实战项目：京东、阿里云、聚合数据、百度糯米等企业网站开发。

第二阶段：


1. JavaScript高级语法、面向对象思想、JavaScript中的对象、JSON、闭包、原型、作用域链、正则表达式；
2. 同步与异步概念、原生Ajax、jQuery的Ajax相关API使用、低层原理分析、缓存问题及处理方式、跨域请求及解决方法、前端模块化（RequireJs）；
3. NodeJs开发环境搭建、NodeJs基础语法、ES6常用语法、核心类库、异步编程、Express框架的使用；
4. jQuery基础语法、工作原理、DOM对象与jQuery对象 熟练掌握jQuery库、能使用jQuery完成各种网页交互与用户处理；
实战项目：亿升财富、智联招聘、乐鱼网等完整的系统开发。

第三阶段：


1. TypeScript环境搭建、TypeScript基础语法、Webpack应用、自动化流程环境构建Gulp，node环境、前端自动化流程构建、Github搭建；
2. Canvas标签基础知识、绘制矩形、绘制圆形、图像裁剪、动画、HTML5小游戏。
实战项目：飞翔小鸟、坦克大战、个人博客系统、水果忍者等游戏开发。

第四阶段：


1. VueJs组件开发介绍、VueJs组件开发实例详解、Vue-router开发SPA应用、Vuex应用状态管理、AngularJs4入门、Angu larJs进阶、AngularJs进阶-Services与指令的使用、AngularJs内置服务$http的使用、AngularJs进阶-Directive的使用、ReactJs框架基本使用、JSX语法及特点介绍
2.混合式APP开发应用（ReactNative/Ionic）；
实战项目：商城移动端项目、学生考试系统、AO管理系统、微信小程序开发等综合实战项目开发。
